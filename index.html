NHL Website

For this project, I designed and developed a comprehensive website for an NHL team. The site features real-time game updates, player statistics, and an engaging fan community forum. Using React for the front-end and integrating with the NHL's API, I ensured fans stay informed with the latest news and scores. The website also includes a merchandise store and ticket purchasing options, providing a seamless experience for supporters. My focus on responsive design ensures that fans can access the site from any device, enhancing their connection with the team.

HR Website

This project involved creating an all-in-one HR management platform aimed at streamlining employee onboarding, performance tracking, and benefits management. Built with Angular and leveraging Node.js for the backend, the platform offers a user-friendly interface for HR professionals and employees alike. Key features include automated payroll processing, leave management, and detailed analytics dashboards. The system supports secure data handling and complies with GDPR regulations. By implementing a modular design, the platform can be easily customized to meet the unique needs of different organizations.

SaaS Application

In this project, I developed a scalable Software-as-a-Service (SaaS) application designed to help small businesses manage their operations efficiently. The application includes modules for inventory management, customer relationship management (CRM), and financial reporting. Utilizing Vue.js for the front-end and Firebase for the backend, the application ensures real-time data synchronization and robust security. The intuitive user interface allows businesses to quickly adapt the software to their workflows, while the cloud-based infrastructure supports seamless updates and scalability. My role involved both front-end development and backend integration, ensuring a cohesive and efficient product.
